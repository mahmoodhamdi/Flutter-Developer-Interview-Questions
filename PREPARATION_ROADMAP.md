# Flutter Interview Preparation Roadmap

## Overview

This roadmap helps you prepare for Flutter developer interviews at different levels. Follow the path that matches your target position.

---

## Junior Developer (0-1 Year Experience)

### Week 1: Dart Fundamentals

**Day 1-2: Basics**
- [ ] Variables, types, null safety
- [ ] Functions and parameters
- [ ] Control flow (if, switch, loops)

**Day 3-4: OOP**
- [ ] Classes and objects
- [ ] Constructors (default, named, factory)
- [ ] Inheritance and mixins

**Day 5-7: Collections & Async**
- [ ] List, Map, Set
- [ ] Future and async/await
- [ ] Error handling (try-catch)

**Practice:** Solve 5-10 easy DSA problems in Dart

### Week 2: Flutter Basics

**Day 1-2: Widgets**
- [ ] StatelessWidget vs StatefulWidget
- [ ] Common widgets (Container, Row, Column, Stack)
- [ ] Widget lifecycle

**Day 3-4: Layout**
- [ ] Flex layout (Expanded, Flexible)
- [ ] Constraints and sizing
- [ ] Responsive design basics

**Day 5-7: Forms & Navigation**
- [ ] Form validation
- [ ] TextFormField and controllers
- [ ] Navigator and routes

**Project:** Build a simple login/register app

### Week 3: State Management & Networking

**Day 1-3: State Management**
- [ ] setState basics
- [ ] Provider package
- [ ] InheritedWidget concept

**Day 4-5: Networking**
- [ ] HTTP package
- [ ] JSON parsing
- [ ] REST API integration

**Day 6-7: Local Storage**
- [ ] SharedPreferences
- [ ] Basic file handling

**Project:** Build a todo app with API

### Week 4: Practice & Review

- [ ] Review all topics
- [ ] Mock interviews
- [ ] Build a portfolio project
- [ ] Practice explaining your code

## Final Tips

1. **Consistency over intensity** - 2 hours daily beats 10 hours once
2. **Build projects** - Theory alone isn't enough
3. **Explain to others** - Teaching solidifies learning
4. **Take breaks** - Burnout hurts performance
5. **Stay updated** - Follow Flutter releases and news
